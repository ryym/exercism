### factor に登場する数字が 1 だけだと必ず回分数になる

```
11 * 11 = 121
111 * 111 = 12,321
1111 * 111111 = 123,444,321
```

101, 1001 とかでもできるっぽい?
10101 でもできるけど、10011 はダメだった。

### 回分数には何か法則がある?

テストに出てくる回分数が似てる: 121, 9009, 10201, 90609

---

### どう実装する？

#### 範囲内の数のペアを試していく

小さい方: 10x11, 10x12, 10x13, ...
大きい方: 99x99, 99x98, 99x97, ...

最初に見つかった回分数の約数を列挙する。

#### 範囲内の数のペアで作れる数の中から先に回分数を探す

10-99 なら最小は10x10, 最大は99x99なので、順に回分になる数を探し、
見つかったらその数を範囲内の数の乗算で求められるかを計算する。
